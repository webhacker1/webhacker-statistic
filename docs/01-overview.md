# Overview

Библиотека `webhacker-statistic` рендерит графики на чистом `canvas` без сторонних сервисов и chart-библиотек.

Экспортируемый API:

- `window.WebHackerStatistics.initStatistic()`
- `window.WebHackerStatistics.statisticTest(containerElement)`

Также остаются совместимые глобальные алиасы:

- `window.initStatistic()`
- `window.statisticTest(containerElement)`
